# 《美悦子系列》 (Miyetsuko Series)
**“在欲望与理智的交界处，凝视深渊的人，终将成为深渊本身。”**
### 📂 档案概述
《美悦子系列》是一部融合了黑色悬疑、心理博弈与暗黑基调的现代长篇虚构小说。故事以极其冷静细腻的笔触，剥开日常生活光鲜平庸的表象，带领读者坠入一个充满控制、背叛、救赎与畸恋的隐秘地下世界。
### 📖 故事梗概
整部系列围绕着谜一样的核心人物——林美悦子展开。她游走于光与影的边缘，宛如一面折射出人性极度渴望与恐惧的多棱镜。
从静谧幽暗的校园琴房，到暗流涌动的都市角落，美悦子如同一个漫不经心的执棋者，将周围的人一步步卷入她精心编织的漩涡。无论是被无形锁链牵绊、在毁灭与重生中挣扎的健太（Kenta），还是在道德边界与失控边缘反复试探的白石（Shiraishi），每一个试图靠近她的人，最终都被迫直面自己内心深处最扭曲的隐秘与疯狂。
这不仅是一场关于权力、肉体与谎言的危险博弈，更是一次对现代人精神困境与原始欲望的残酷解剖。
### 核心主题
 * **面具与真实**：每个人都在扮演社会赋予的安全角色，而美悦子是那个残忍撕下面具的引路人。
 * **控制与臣服**：在看似偶然的相遇背后，权力的高低位在无形中不断反转。
 * **危险的浪漫**：将极度的理性与失控的欲望交织，书写出一种令人窒息却又无法抗拒的黑色美学。
